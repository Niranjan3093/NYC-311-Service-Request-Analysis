4. Statistical Testing¶

4.1. Test 1: Whether the average response time across complaint types is similar or not.¶

● State the Null Hypothesis (H0) and Alternate Hypothesis (H1).¶

● Perform the statistical test and provide the p-value.¶

● Interpret the results to accept or reject the Null Hypothesis.¶

Code:

In [648]:
from scipy.stats import f_oneway

# Convert the "Request_Closing_Time" timedelta column into hours (float) for analysis
df_cleaned.loc[:, 'Request_Closing_Hours'] = (
    df_cleaned['Request_Closing_Time'].dt.total_seconds() / 3600
)

# Identify the top 3 most frequent complaint types for comparison
top3 = df_cleaned['Complaint Type'].value_counts().nlargest(3).index.tolist()

# Extract the closing time data (in hours) for each of the top 3 complaint types
# Each group's data will be used as a sample in the ANOVA test
groups = [
    df_cleaned.loc[df_cleaned['Complaint Type'] == c, 'Request_Closing_Hours']
    for c in top3
]

# Perform one-way ANOVA to test if the average response times differ across complaint types
F_stat, p_val = f_oneway(*groups)

# Output the F-statistic and p-value to interpret the test result
print(f"ANOVA result: F = {F_stat:.3f}, p = {p_val:.3f}")

ANOVA result: F = 804.923, p = 0.000

from scipy.stats import chi2_contingency
import pandas as pd

# Create a contingency table showing the frequency of each complaint type across boroughs
# Rows: Complaint Types | Columns: Boroughs | Values: Count of occurrences
contingency = pd.crosstab(df_cleaned['Complaint Type'], df_cleaned['Borough'])

# Perform the chi-square test of independence to determine if complaint type is related to borough
chi2, p_val, dof, expected = chi2_contingency(contingency)

# Print the chi-square statistic, p-value, and degrees of freedom
print(f"Chi-square result: χ² = {chi2:.2f}, p = {p_val:.3f}, dof = {dof}")
